# crestron-magicpacket

SIMPL# Pro class to power on devices using [Wake-on-LAN](https://en.wikipedia.org/wiki/Wake-on-LAN)


## Usage
```c#
WakeOnLAN WOL = new WakeOnLAN();
WOL.WakeUp(mac);
```
